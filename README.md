# caps-rate

Get how much a string IS YELLING.

## Example

``` javascript
const caps = require('caps-rate');

caps('hello WORLD');
// => 0.45454545454545453
```

## Installation

``` bash
$ npm install caps-rate
```

## API

``` javascript
const caps = require('caps-rate');
```

### caps(value)

Returns how frequently caps are used in `value`.
